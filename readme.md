This project is a **hands-on demonstration** of how a blockchain works.

Instead of learning blockchain only through theory, this project allows you to:

* Observe how information becomes **permanent and publicly readable**

## ðŸ§  What does this project do?

The project simulates a **public identity ledger**.

* Users can submit a simple identity entry (like a name)
* Each submission is recorded as a **real blockchain transaction**
* Transactions are grouped into **blocks**
* Once added, entries **cannot be edited or removed**
* Anyone can view the **entire history of entries**

There are **no accounts, wallets, or tokens involved** â€” only data and history.
