# Registration Task

## Objective
Write a script to interact with the blockchain and call a Smart Contract function.

## Requirements
1.  **Connect** to the Host's RPC URL (Ask Host for IP).
2.  **Authenticate** using your Private Key (Import it as a Wallet).
3.  **Interact** with the `StudentRegistry` contract (Address provided by Host).
4.  **Execute** the `register(name, rollNumber)` function.
